# Configurar LAMP Amazon Linux 2 y WSGI 

Script de configuración LAMP Amazon Linux 2 con WSGI (Python3 - Flask).

## Ejecutar el script:
#### Paso 1:
Otorgar permisos de ejecución.
>>>
chmod u+xr config_AMI_WSGI.sh
>>>
#### Paso 2:
Correr el script.
>>>
sudo ./config_AMI_WSGI.sh
>>>

## Referencias:
